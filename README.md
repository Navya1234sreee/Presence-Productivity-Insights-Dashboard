# Presence-Productivity-Insights-Dashboard
Project Overview: Presence & Productivity Insights
This project involved developing a comprehensive HR Analytics Dashboard designed to monitor and analyze workforce engagement through three primary lenses: physical/digital presence, remote work adoption, and leave patterns.
1. Objective

   
The goal was to create a data-driven tool that provides leadership with a clear view of how the workforce is operating in a hybrid environment. It focuses on identifying attendance trends, quantifying the impact of Work From Home (WFH) policies, and monitoring health-related risks through Sick Leave (SL) tracking.



This dashboard, titled Presence & Productivity Insights, provides a comprehensive look at workforce engagement and leave patterns. Below are the key points and a draft LinkedIn post you can use to showcase this work.
